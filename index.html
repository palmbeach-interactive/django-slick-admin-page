Django Slick Admin

A slick & clean look for your Django admin interface
Currently v0.1.5

Preview Github

Why another Django Admin?

With Slick Admin we aim to provide a consistent, modern looking and customisable admin interface for standard views and widgets of the current django admin (>=1.9).

Slick Admin provides you with a cosistent looking admin interface with or without using django-cms (our preferred CMS) in your django application. No need to install a CMS just to have similar looking interface for all your django based projects.

Slick Admin works completely unobtrusive and does everything via CSS. Well not everything - there are some optional & tiny bits of JavaScript involved as well.

Slick Admin works with the standard Django admin - and comes with support for:

  • django CMS
  • Django Filer
  • Django Admin Tools
  • Django Admin Tools Dashboard
  • django-hvad
  • django-parler

Admin Comparsion

All features of the standard django admin interface are always available. Django Slick Admin focuses on a seamless and unobstrusive integration of often used django apps like model language tabs, expandable inline elements etc. The Dashboard is not mandatory but a helpful way to organise more complex django applications.

Easy Customisation & Branding

Django Slick Admin can be easily adjusted to reflect your projects or your clients visual needs.

Support for Material Icons

All Material Design Icons can be used to customize your projects dashboard.

Installation

Using the latest version from PyPI:

pip install django-slick-admin

Using this GitHub repository (latest):

pip install -e "git://github.com/palmbeach-interactive/django-slick-admin.git#egg=django-slick-admin"

More detailled instructions and help integrating django-slick admin with 3rd party apps like django-admin-tools can be found on Github.
